📦 Object Sorting Conveyor

A PLC-based Object Detection and Sorting Project using S7-300

📘 Project Overview

This PLC project is designed to sort objects based on their size (small, medium, and large) using sensors and actuators. The system uses Siemens S7-300 PLC and includes timers, memory bits, and pusher outputs to sort the objects at the correct stations.

🟢 Network 1: Start Logic of Conveyor

  • M0.0 – "Start PB" input push button.
  • M0.1 – "Start bit" (internal memory) set when Start PB is pressed.
  • M0.1 latches and is used throughout the program as system ON indicator.

🟡 Network 2: Detect Small Size Object

  • M0.1 – System ON condition.
  • I0.0 – Small size sensor must be ON.
  • I0.1 – Medium sensor must be OFF.
  • I0.2 – Large sensor must be OFF.
  • T1 – ON Delay Timer to avoid false triggering.
  • I0.5 – Confirm pusher not already diverted.
  • M0.2 – Memory bit to store small object detection.

🟠 Network 3: Detect Medium Size Object

  • All sensors I0.0, I0.1, I0.2 are checked.
  • Only Medium sensor I0.1 should be ON; others OFF.
  • T2 is triggered and starts T4 (medium timer).
  • M0.3 – Memory bit for medium object detection.

🔴 Network 4: Detect Large Size Object

  • M0.1 – System ON.
  • Large sensor I0.2 must be ON, others OFF.
  • T2 and T5 used for large delay timing.
  • M0.4 – Large object detected memory bit.

🔁 Network 5: Logic to Divert Medium Object

  • M0.1 – System ON.
  • M0.3 – Medium object detected.
  • I0.4 – Medium object reaches pusher area.
  • Q0.2 – Activates medium object pusher.
  • T4 – Timing control for pusher operation.

🔁 Network 6: Logic to Divert Large Object

  • M0.1 – System ON.
  • M0.4 – Large object detected.
  • I0.3 – Large object reaches diversion area.
  • Q0.1 – Activates large object pusher.
  • T5 – Time delay for actuator.
